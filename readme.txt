== Description ==

Product bundles for WooCommerce - is a plugin that helps you to create a new type of product in WooCommerce. It will include several products or services as a combined product or service package.

With this plugin, you can bundle assemble products or services as a group and offer a discount. As a result, you can promote customer to buy more to boost your sales.

FEATURES
* Allow to create product as bundles
* Compatible with WordPress
* Compatible with WooCommerce
* Allow to choose item to display products
* Control the number of product displaying
* Fully responsive

== Installation ==

How to install the plugin and get it working?

1. Upload the plugin files to the `/wp-content/plugins/wp_fullpage` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Create a product and choose HMH
